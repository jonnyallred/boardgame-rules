# Mixmo

## Overview

Mixmo is a fast-paced word game where all players simultaneously race to form crossword grids from letter tiles. Players start with a small set of tiles and try to arrange them into valid crosswords. When one player succeeds, all players draw additional tiles and must incorporate them into their grids. The game continues until the tile pool is exhausted, and the first player to use all their tiles in a valid crossword wins.

## Components

- Letter tiles (similar to Scrabble-style tiles)
- Tile bag or pool

## Setup

1. Place all letter tiles face down in the center of the table.
2. Each player draws 6 tiles.
3. At the signal, all players simultaneously begin.

## Turn Structure

Play is simultaneous and continuous:
1. All players work at the same time to form crossword grids.
2. When any player completes a valid crossword using all their tiles, they call out.
3. All players draw additional tiles (typically 2 more).
4. Players must rearrange their grids to incorporate new tiles.
5. Repeat until the tile pool is empty.

## Actions

### Building Crosswords
- Arrange tiles into a valid crossword grid (interconnected words reading left-to-right and top-to-bottom).
- All words must be valid dictionary words.
- Every tile must be part of the grid.
- You may rearrange your entire grid at any time.

### Drawing Tiles
- When any player completes their grid, everyone draws more tiles.
- New tiles must be incorporated into the crossword.

## Scoring / Victory Conditions

The game ends when the tile pool is empty and one player successfully uses all their tiles in a valid crossword grid. That player wins.

If no player can complete a valid grid after the pool is exhausted, the player with the fewest unused tiles wins.

## Special Rules & Edge Cases

- All play is simultaneous; there are no turns.
- Players may completely rearrange their grids at any time.
- All words must be valid (no proper nouns, abbreviations, etc.).
- Speed and anagram skills are key.

## Player Reference

**Play:** Simultaneously build crossword grids -> When any player finishes, all draw more tiles -> Repeat until pool empty

**Win:** First to use all tiles in a valid crossword after pool is empty
