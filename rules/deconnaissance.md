## Overview

Deconnaissance is a trivia/quiz board game where players race across the board by answering questions correctly. Each question provides the theme of the answer, a definition, and an optional hint if players cannot find the answer. The first player to reach the other end of the board wins. The game tests general knowledge and vocabulary in a competitive format.

## Components

- 1 game board with a path from start to finish
- Question cards (with theme, definition, and hint)
- Player pawns
- Dice or movement mechanism
- Rules booklet

## Setup

1. Place the board in the center of the table.
2. Each player places their pawn at the start position.
3. Shuffle the question cards and place them in a draw pile.
4. Determine the starting player.

## Turn Structure

On your turn:
1. **Draw a Question:** Another player draws a question card and reads the theme and definition.
2. **Answer:** Attempt to give the correct answer.
3. **Hint (Optional):** If you cannot answer, request the hint for a second chance.
4. **Move:** If you answer correctly, advance your pawn along the board. Incorrect answers mean no movement.

## Actions

### Answering Questions
- Listen to the theme and definition read from the question card.
- Provide your answer. If correct, move forward.

### Requesting a Hint
- If you cannot answer from the theme and definition alone, request the hint.
- The hint provides additional information to help identify the answer.
- Answering correctly after using a hint may result in reduced movement (variant rules).

### Movement
- Advance your pawn along the board path when you answer correctly.
- The distance moved may depend on difficulty or whether a hint was used.

## Scoring / Victory Conditions

The first player to reach the end of the board path wins the game.

## Special Rules & Edge Cases

- **Three-Tier Clues:** Each question provides escalating help: theme, definition, then hint. This graduated difficulty helps players of different knowledge levels compete.
- **French Language:** The game is primarily in French; questions and answers are in French.
- **Knowledge Categories:** Questions span various topics, testing breadth of general knowledge.

## Player Reference

**Turn:** Question read (theme + definition) → Answer → Optional hint → Move if correct

**Win:** First to reach the end of the board

**Clue Levels:** Theme → Definition → Hint (escalating help)
