# Monty Python Fluxx

## Overview

Monty Python Fluxx is an ever-changing card game themed around Monty Python sketches and films. The game begins with simple rules (draw 1, play 1) but players add New Rules, Goals, Keepers, and Actions that constantly change how the game is played. The first player whose collection of Keepers matches the current Goal wins. This Fluxx version adds Creepers that attach to players and themed Monty Python cards.

## Components

- Deck of cards including:
  - Basic Rules card (always in play)
  - New Rules cards
  - Goal cards
  - Keeper cards (characters and objects from Monty Python)
  - Action cards
  - Creeper cards

## Setup

1. Place the Basic Rules card in the center of the table.
2. Shuffle the deck and deal 3 cards to each player.
3. Place the remainder face down as the draw pile.
4. The dealer goes first. Play proceeds clockwise.

## Turn Structure

Each turn follows three steps:

1. **Draw** the number of cards currently required.
2. **Play** the number of cards currently required.
3. **Discard** to comply with any Limit rules in play.

Optional actions allowed by New Rules (like free plays) may be taken at any point during your turn.

## Actions

### Card Types

| Type | Effect |
|------|--------|
| **New Rule** | Place face up in center. Takes effect immediately. If it contradicts an existing rule, the old rule is discarded |
| **Goal** | Place face up in center. Replace any existing Goal. Shows which Keepers you need to win |
| **Keeper** | Play face up in front of you. Keepers are what you need to match the Goal |
| **Action** | Do what the card says, then discard it. Actions are single-use |
| **Creeper** | When drawn, immediately place in front of you and draw a replacement. Creepers generally prevent you from winning unless the Goal specifically requires them |

### Basic Rules

- **Draw 1**: Draw 1 card per turn.
- **Play 1**: Play 1 card per turn.
- New Rules may change these to Draw 2/3/4/5 or Play 2/3/4/5, add Hand/Keeper Limits, etc.

### Limits

When a Limit rule is played (Hand Limit or Keeper Limit), all players must immediately comply, discarding excess cards/Keepers of their choice.

## Scoring / Victory Conditions

The game continues until someone meets the conditions of the current Goal. A Goal typically requires having specific Keepers in front of you. The player who meets the Goal wins instantly, regardless of whose turn it is (including ties -- if a tie, the game continues until only one winner emerges).

## Special Rules & Edge Cases

- All cards are playable even if they have no effect. If unsure how a card works, read the full text.
- If you must draw more cards during your turn (e.g., a New Rule increases the draw count), draw the additional cards immediately.
- Creepers cannot be held in hand. As soon as you draw one, play it and draw again (this does NOT count as a "play").
- You may accept multiple Creepers as you draw.
- **First Round Creepers**: At the start of the game, if any player's initial hand contains a Creeper, they must immediately place it and draw a replacement.
- **Reshuffling**: If the draw pile is exhausted, shuffle the discard pile to form a new draw pile.
- **Dropping Out**: Players can leave at any time by discarding their hand and Keepers.
- Monty Python-themed cards reference specific sketches and add unique interactions consistent with Fluxx rules.

## Player Reference

| Phase | Action |
|-------|--------|
| Draw | Draw cards equal to current Draw rule |
| Play | Play cards equal to current Play rule |
| Discard | Comply with Hand/Keeper Limits |
| Win | Match the current Goal with your Keepers |
