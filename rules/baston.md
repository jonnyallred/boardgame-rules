## Overview

Baston is a humorous skirmish fighting board-wargame set in a suburban French bar. Each player manages a gang of bad boys fighting with bottles of beer and motorcycle chains. The game combines tactical combat with a lighthearted, comedic theme as players brawl for dominance in the bar.

## Components

- Game board (depicting a French bar interior)
- Gang member figures/counters
- Weapon tokens (beer bottles, chains, etc.)
- Furniture pieces (tables, chairs as obstacles/weapons)
- Combat cards or dice
- Rules booklet

## Setup

1. Place the game board showing the bar layout.
2. Each player selects a gang and takes the corresponding figures.
3. Place gang members in starting positions on opposite sides of the bar.
4. Distribute weapon tokens and set up furniture obstacles.

## Turn Structure

Players alternate turns:
1. **Movement:** Move gang members through the bar.
2. **Action:** Attack opponents, pick up weapons, throw furniture.
3. **Combat Resolution:** Resolve any fights that occur.

## Actions

- **Move:** Navigate gang members through the bar, around furniture.
- **Melee attack:** Use fists, beer bottles, or chains to attack adjacent opponents.
- **Throw objects:** Hurl bottles or furniture at distant opponents.
- **Pick up weapons:** Grab items from the bar environment.
- **Take cover:** Use furniture for protection.

## Scoring / Victory Conditions

The last gang standing (or the gang with the most members still conscious) wins the bar brawl.

## Special Rules & Edge Cases

- **Furniture interaction:** Tables and chairs can be used as weapons or barriers.
- **Beer bottles:** Can be used as melee weapons or thrown projectiles.
- **Humorous theme:** The game emphasizes fun and chaos over strict tactical accuracy.
- **French-language:** Originally published in French.

## Player Reference
