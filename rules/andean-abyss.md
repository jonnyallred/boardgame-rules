## Overview

Andean Abyss is a COIN (Counter-Insurgency) series game simulating the multi-faction conflict in Colombia during the 1990s and 2000s. Four factions -- Government, FARC, AUC (paramilitaries), and Cartels -- each have unique victory conditions, capabilities, and special activities. Players execute operations and special activities driven by an event card system that determines faction order. The game combines area control, asymmetric powers, and political maneuvering.

## Components

- 1 Map of Colombia (departments and cities)
- Event cards deck (72 cards)
- Faction pieces: Government (troops, police, bases), FARC (guerrillas, bases), AUC (guerrillas, bases), Cartels (guerrillas, bases)
- Eligibility cylinders
- Aid, Resources, and Support/Opposition markers
- Propaganda cards (embedded in deck)
- Various tracking markers
- Sequence of Play aid cards

## Setup

1. Place the map centrally. Set up faction pieces according to the scenario instructions.
2. Set initial resource levels for each faction.
3. Place support/opposition markers in departments as indicated.
4. Shuffle event cards with propaganda cards inserted at specified intervals.
5. Place eligibility markers for all factions.

## Turn Structure

Each card turned reveals an event with faction order shown. Factions act in the listed order, choosing from:

1. **Execute the Event** (as described on the card)
2. **Conduct an Operation** (+ optional Special Activity)
3. **Pass** (gain resources)

Only the first 2 eligible factions may act per card. After acting, a faction becomes ineligible for the next card. Passing keeps you eligible.

## Actions

### Government Operations

| Operation | Effect |
|-----------|--------|
| Train | Place troops/police, build bases, civic action (shift support) |
| Patrol | Move troops along lines of communication, activate for security |
| Sweep | Move troops into departments to find hidden guerrillas |
| Assault | Remove enemy pieces in departments with troops |

**Special Activities:** Eradicate (destroy drug crops), Air Strike, Air Lift

### FARC Operations

| Operation | Effect |
|-----------|--------|
| Rally | Place guerrillas, build bases |
| March | Move guerrillas between departments |
| Attack | Remove enemy pieces |
| Terror | Place terror markers, shift opposition |

**Special Activities:** Kidnap (gain resources), Extort, Ambush

### AUC Operations

| Operation | Effect |
|-----------|--------|
| Rally | Place guerrillas, build bases |
| March | Move guerrillas |
| Attack | Remove enemy pieces |
| Terror | Place terror, shift opposition |

**Special Activities:** Assassinate, Extort

### Cartel Operations

| Operation | Effect |
|-----------|--------|
| Rally | Place guerrillas, build bases, cultivate (place drug crops) |
| March | Move guerrillas |
| Attack | Remove enemy pieces |
| Terror | Place terror |

**Special Activities:** Bribe (flip enemy pieces), Cultivate, Process

## Scoring / Victory Conditions

When a Propaganda card is drawn, check victory conditions:

| Faction | Victory Condition |
|---------|------------------|
| Government | Support + Aid exceeds threshold |
| FARC | Opposition + FARC bases exceeds threshold |
| AUC | AUC bases in departments with no FARC bases |
| Cartels | Resources + bases exceeds threshold |

If no faction wins at the final Propaganda card, the faction closest to their condition wins.

## Special Rules & Edge Cases

- **Hidden Guerrillas:** Guerrillas start hidden (underground). Sweeping reveals them. Only revealed guerrillas can be assaulted.
- **Lines of Communication (LoCs):** Roads connecting departments; Patrol and economic effects use LoCs.
- **Population:** Only populated departments can have support/opposition.
- **Propaganda Rounds:** Reset eligibility, collect resources, redistribute aid, check victory.
- **Solo/2-Player:** Non-player factions use flowchart-based bot systems.

## Player Reference

**Card Flow:** Reveal card → 1st/2nd eligible factions choose: Event / Operation+Special / Pass → Adjust eligibility

**Faction Resources:** Government (Aid), FARC/AUC (Extortion), Cartels (Drug revenue)

**Key Mechanic:** Guerrillas are hidden until swept; only revealed pieces can be attacked.
