## Overview

Bug Hunter (full title: Bug Hunter Sniper!) is SPI's man-to-man tactical wargame that takes the familiar infantry combat system into a science fiction setting, with an Aliens-inspired flavor. Players control individual soldiers or alien creatures in close-quarters tactical combat, using detailed rules for line of sight, weapon fire, and movement through corridors and rooms.

## Components

- Game map (interior corridors and rooms)
- Die-cut counters (human soldiers and alien "bugs")
- Dice
- Rules booklet
- Charts and tables

## Setup

1. Select a scenario from the scenario booklet.
2. Place the map between players.
3. One player takes the human soldier counters, the other takes the alien bug counters.
4. Deploy units according to scenario setup instructions.

## Turn Structure

Each turn:
1. **Initiative Phase:** Determine which player acts first.
2. **First Player Phase:** Activate and move units, then fire weapons.
3. **Second Player Phase:** Activate and move units, then fire weapons.
4. **Close Combat Phase:** Resolve any melee engagements.

## Actions

**Move:**
- Individual soldiers/bugs move a number of spaces based on their movement allowance.
- Terrain (walls, doors, corridors) restricts movement and provides cover.

**Fire Weapons:**
- Human soldiers use ranged weapons (rifles, automatic weapons, grenades).
- Check line of sight to target.
- Roll dice and compare against weapon accuracy tables modified by range, cover, and other factors.

**Close Combat:**
- When units are in adjacent spaces, melee combat may occur.
- Bugs are typically stronger in close quarters; humans prefer ranged engagement.

**Special Actions:**
- Open/close doors, set ambushes, overwatch (opportunity fire on enemy movement).

## Scoring / Victory Conditions

Victory conditions vary by scenario. Typical objectives include eliminating all enemy units, defending a position for a set number of turns, or extracting to a designated area.

## Special Rules & Edge Cases

- **Line of Sight:** Detailed LOS rules determine what each unit can see and target. Corridors and rooms create natural chokepoints.
- **Overwatch:** Units can be placed in overwatch mode, allowing them to fire during the enemy's movement phase.
- **Alien Abilities:** Bug creatures may have special abilities like acid blood, enhanced speed, or the ability to move through vents.
- **Man-to-Man Scale:** Each counter represents a single individual, making the game highly tactical and detailed.
- **Sci-Fi Setting:** While using SPI's familiar infantry combat system, the alien opposition creates asymmetric gameplay.
