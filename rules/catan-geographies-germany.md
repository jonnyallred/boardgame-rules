## Overview

Catan Geographies: Germany is a scenario overlay for the Catan base game that replaces the island of Catan with a fixed map of Germany. Players settle the regions of Germany, building roads, settlements, and cities across familiar German geography. The scenario uses the standard Catan resource and building system with a pre-set map featuring real German cities and regions. Special landmark tokens provide additional victory point opportunities.

## Components

- 1 large fixed game board depicting Germany with terrain hexes
- Number tokens
- Landmark tokens (representing famous German sites)
- Uses standard Catan resource cards, development cards, and player pieces (not included — requires Catan base game)

## Setup

Place the Germany board (fixed map, no variable setup). Place number tokens as indicated. Each player places starting settlements and roads on the pre-set map. Place landmark tokens on designated locations.

## Turn Structure

Standard Catan turn structure: roll dice, trade, build. The core rules are unchanged from base Catan.

## Actions

Standard Catan building costs and actions. Additionally, when you build a settlement or city adjacent to a landmark, you may claim the landmark token for bonus victory points.

## Scoring / Victory Conditions

First to 12 victory points wins (higher than standard Catan to account for landmark bonuses). Standard VP sources plus landmark tokens.

## Special Rules & Edge Cases

- Requires the Catan base game for resource cards, development cards, and player pieces.
- The map is fixed (no variable hex placement).
- Landmark tokens add a geographic discovery element to the game.
- All other standard Catan rules apply.

## Player Reference

Standard Catan building costs apply. Win condition: 12 VP (includes landmark bonuses).
