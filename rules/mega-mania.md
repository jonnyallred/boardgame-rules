# Mega-Mania

## Overview

Block Mania: Mega-Mania is a supplement to Block Mania that expands the game from 2 to 4 players. Set in the Judge Dredd universe, players control the residents of city blocks who must cause as much damage as possible to opposing blocks before the Judges arrive to restore order. The player whose block is least damaged at the end wins.

## Components

- Additional game boards (2 more city blocks for 4-player games)
- Additional unit counters
- Mania cards (game timer deck)
- Command point dice
- Combat resolution tables
- Rules supplement

## Setup

1. Set up 3 or 4 city block boards (2 from base game + Mega-Mania expansion blocks).
2. Each player takes a city block and its resident units.
3. Prepare the Mania card deck as the game timer.
4. Each player places starting units in their block.

## Turn Structure

Each round:
1. **Command Phase:** Roll 2d6 to determine command points for the turn.
2. **Action Phase:** Spend command points to recruit units, move, and perform actions.
3. **Combat Phase:** Resolve attacks (shooting, melee, arson, etc.).
4. **Mania Phase:** Draw a Mania card into the active player's hand. When the deck is exhausted, the Judges arrive and the game ends.

## Actions

Actions cost varying amounts of command points:

| Action | Description |
|--------|-------------|
| **Recruit** | Bring new units into play in your block |
| **Move** | Move units between locations |
| **Shoot** | Ranged attack against enemy units or buildings |
| **Close Combat** | Melee attack against adjacent enemies |
| **Loot** | Steal resources from opponent's block |
| **Arson** | Set fires in opponent's block |
| **Scrawl** | Graffiti opponent's block (cosmetic damage) |
| **Fire Fighting** | Extinguish fires in your own block |

## Scoring / Victory Conditions

When the Mania card deck is exhausted, the Judges arrive and the game ends. The player whose city block has sustained the **least damage** wins.

Damage is tracked per block and includes structural damage, fires, and casualties.

## Special Rules & Edge Cases

- Mega-Mania expands Block Mania from 2 to 4 players; the base game rules apply.
- Each block has unique characteristics and starting conditions.
- Mania cards can also trigger special events when drawn.
- Some units have special abilities (snipers, heavies, etc.).
- The game requires the Block Mania base game to play.

## Player Reference

**Turn:** Roll command points (2d6) -> Recruit/Move/Attack -> Draw Mania card

**Combat types:** Shooting, Close Combat, Arson, Looting

**Win condition:** Least-damaged block when Judges arrive (Mania deck empty)
