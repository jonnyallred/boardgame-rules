## Overview

First Orchard (Erster Obstgarten / My Very First Games - First Orchard) is a cooperative game for 1-4 players ages 2 and up, designed by HABA. Players work together to harvest all the fruit from the trees before the raven reaches the orchard. It is a simplified version of the classic HABA game Orchard, designed as a first board game experience for toddlers, teaching colors, taking turns, and cooperative play.

## Components

- 1 Raven figure
- 1 Fruit basket
- 16 Wooden fruits (4 green apples, 4 red cherries/apples, 4 yellow pears, 4 blue plums)
- 4 Trees (cardboard)
- 5 Path cards
- 1 Color-symbol die (showing 4 colors, a basket, and a raven)
- Instructions

## Setup

1. Place the 4 trees in the center of the play area.
2. Place the matching colored fruits on each tree (4 fruits per tree).
3. Lay the 5 path cards in a row next to the trees, forming the raven's path.
4. Place the raven before the first path card.
5. Place the basket nearby where all players can reach it.
6. Place the die within reach.

## Turn Structure

Youngest player (or whoever most recently ate an apple) goes first. Play continues clockwise. On each turn, the active player rolls the die once and performs the corresponding action.

### Die Results

- **Color face (green, red, yellow, or blue)**: Pick one fruit of that color from its tree and place it in the basket.
- **Basket symbol**: Pick any one fruit from any tree and place it in the basket.
- **Raven symbol**: Move the raven forward one path card.

If a color is rolled but no fruits of that color remain on the tree, nothing happens — pass the die to the next player.

## Actions

### Harvesting Fruit

When a color is rolled, take one matching fruit from the corresponding tree and place it in the basket. When the basket symbol is rolled, choose any available fruit from any tree.

### Raven Movement

When the raven symbol is rolled, move the raven one path card forward along the path toward the orchard.

## Scoring / Victory Conditions

**Win (cooperative)**: If all 16 fruits are harvested and placed in the basket before the raven passes the last path card, all players win together.

**Lose (cooperative)**: If the raven passes the 5th and final path card and reaches the orchard before all fruits have been picked, all players lose together.

## Special Rules & Edge Cases

- **No Fruit Available**: If a color is rolled and no fruits of that color remain, the player's turn is simply skipped (no action taken). Pass the die.
- **Basket Roll**: The basket symbol lets you choose any fruit, making it a helpful roll.
- **Cooperative Game**: There is no individual winner; all players win or lose together.
- **Solo Play**: The game can be played solo, with the single player taking all turns.
- **Age Range**: Designed for ages 2+, with extra-large wooden pieces safe for toddlers.

## Player Reference

### Turn Sequence
1. Roll the die
2. Perform the action shown (pick a colored fruit, pick any fruit, or move the raven)
3. Pass the die clockwise

### Die Faces
| Symbol | Action |
|--------|--------|
| Green | Pick 1 green fruit |
| Red | Pick 1 red fruit |
| Yellow | Pick 1 yellow fruit |
| Blue | Pick 1 blue fruit |
| Basket | Pick any 1 fruit |
| Raven | Move raven 1 space forward |

### Victory
- Win: All 16 fruits in the basket before raven reaches the orchard
- Lose: Raven reaches the orchard (passes 5th path card) with fruits remaining
