# Mister X

## Overview

Mister X is a modernized version of Ravensburger's classic Scotland Yard, which transfers the manhunt from London to the European continent. One player takes the role of the fugitive Mister X who moves secretly across Europe, while the other players are detectives trying to catch him. Mister X reveals his location periodically, and detectives use transportation networks to deduce his position and corner him.

## Components

- 1 game board (European city map with transport network)
- Mister X figure and screen
- Detective figures
- Transportation tickets (taxi, bus, metro, etc.)
- Movement log for Mister X
- Dice or cards for movement

## Setup

1. Place the board. One player is designated as Mister X.
2. Mister X secretly chooses a starting location and notes it on the log.
3. Detectives place their figures on assigned starting locations.
4. Distribute transportation tickets.

## Turn Structure

1. **Mister X moves** secretly, recording the transport type used.
2. **Detectives move** their figures using transportation tickets.
3. At certain intervals, Mister X must reveal their current location.

## Actions

### Mister X
- Move to any connected location using an available transport type.
- Record the transport used (but not the destination).
- Reveal location at designated intervals (e.g., every 5 turns).

### Detectives
- Move to a connected location using a transportation ticket.
- Coordinate with other detectives to narrow down Mister X's possible positions.
- If a detective lands on Mister X's location, Mister X is caught.

## Scoring / Victory Conditions

- **Detectives win** if any detective lands on Mister X's current location.
- **Mister X wins** if they evade capture for the entire game (survive all rounds).

## Special Rules & Edge Cases

- Mister X has limited special movement options (double moves, secret passages).
- Detectives have limited tickets for each transport type.
- Detectives cannot occupy the same space as each other.
- The game is based on deduction and cooperative play among the detective team.

## Player Reference

**Mister X:** Move secretly, log transport type, reveal location periodically

**Detectives:** Move using tickets, deduce Mister X's position, coordinate to corner

**Win (Detectives):** Land on Mister X | **Win (Mister X):** Survive all rounds
