## Overview

Krypto is a mathematical card game where players race to combine five number cards using arithmetic operations to reach a target number. The first player to find a valid solution wins the round. It is both an educational tool and a competitive game of mental math.

## Components

- 56 Krypto cards numbered as follows:
  - 3 copies each of numbers 1-6
  - 4 copies each of numbers 7-10
  - 2 copies each of numbers 11-17
  - 1 copy each of numbers 18-25

## Setup

1. Shuffle the Krypto deck thoroughly.
2. Deal 5 cards face-up in a row — these are the number cards players must use.
3. Deal 1 card face-up above the row — this is the **target number** (objective).

## Turn Structure

All players work simultaneously. There are no individual turns. Players mentally attempt to combine the five number cards to equal the target number. The first player to find a valid solution calls out and presents it.

## Actions

### Solving
- Use all five number cards exactly once each.
- Combine them using any combination of addition (+), subtraction (-), multiplication (x), and division (÷).
- The five numbers may be arranged in any order.
- Parentheses may be used freely to group operations.
- No other numbers or mathematical symbols (square roots, factorials, etc.) are allowed.

### Winning a Round
- The first player to announce a correct solution wins the round.
- If the solution is verified as correct, that player scores a point (or takes the target card as a trophy).
- If the announced solution is incorrect, that player is eliminated from the current round and others continue.

## Scoring / Victory Conditions

- Play a set number of rounds. The player who wins the most rounds is the overall winner.
- Alternatively, the first player to reach a predetermined number of wins is the champion.

## Special Rules & Edge Cases

- All five number cards must be used exactly once — no more, no less.
- Only the four basic arithmetic operations are permitted.
- Some combinations may have no valid solution; if all players agree no solution exists, deal a new set.
- Division must result in whole numbers (no fractions or decimals in intermediate steps, depending on house rules).
- Multiple valid solutions may exist for the same set of cards.

## Player Reference

**Deal**: 5 number cards + 1 target card

**Goal**: Combine all 5 numbers using +, -, x, ÷ to equal the target

**Rules**: Use each number exactly once. Any order. Parentheses allowed. No other operations.

**Example**: Cards: 3, 6, 11, 14, 15 → Target: 8 → Solution: (14 - 6) x (15 - 11 - 3) = 8
