## Overview

Deckscape: Dracula's Castle is a cooperative card-based escape room game for 1-6 players. Players must solve puzzles and riddles to escape from Dracula's castle. Like all Deckscape games, the entire experience is contained in a single deck of numbered cards with no separate rulebook. Players solve puzzles sequentially, checking answers by flipping cards, with errors tracked for final scoring.

## Components

- 1 Deck of cards (numbered sequentially)
- Players provide: paper, pencil, and a clock/timer

## Setup

1. Remove the deck from the box without shuffling.
2. Prepare paper, pencil, and a timer.
3. Start the timer and begin reading card #1.

## Turn Structure

Cooperative play with no individual turns. The group reads cards in order, solves puzzles together, and flips cards to check answers.

## Actions

### Solving Puzzles
- Read each puzzle card and work together to find the answer.
- Flip the card to check. Correct answers advance the story; incorrect answers add an X to the error count.

### Using Items
- Item cards are set aside when found and used when later puzzles reference them.

### Making Choices
- Some cards present choices that affect the path through the story.

## Scoring / Victory Conditions

Performance is rated based on time taken and number of errors. Fewer errors and faster completion yield a better score.

## Special Rules & Edge Cases

- Designed for a single play-through.
- Cards must remain in numbered order.
- Uses the same core Deckscape system as other titles in the series.

## Player Reference

**Flow:** Read card → Solve puzzle → Flip to check → Next card

**Errors:** X per wrong answer; affects final score

**Score:** Time + errors determine rating
