# Ramses II

## Overview

A memory and deduction game for 2-4 players set in ancient Egypt. Players search for treasures hidden beneath pyramids. On each turn, a search card reveals which treasure to find. Players move pyramids to reveal what is hidden underneath, trying to remember where treasures are located. The player who collects the most valuable search cards wins.

## Components

| Component | Quantity |
|---|---|
| Game board | 1 |
| Pyramids (covering treasures) | 30 |
| Treasure tokens (hidden under pyramids) | 30 |
| Search cards | Multiple |
| Rules booklet | 1 |

## Setup

1. Place the game board in the center.
2. Randomly place treasure tokens on the board spaces.
3. Cover each treasure with a pyramid.
4. Shuffle search cards and place face-down as a draw pile.

## Turn Structure

1. Reveal the top search card (shows a treasure to find).
2. Move pyramids to locate the matching treasure.
3. If found, collect the search card.
4. If not, replace pyramids and the next player tries.

## Actions

### Searching for Treasure
When a search card is revealed, the active player may move a pyramid to peek underneath. Movement rules limit how pyramids can be shifted. If the treasure matches the search card, the player claims the card. Players must remember the positions of treasures they've previously seen.

### Moving Pyramids
Pyramids slide along rows or columns. When a pyramid is moved, it displaces other pyramids in the row. This means the board configuration changes each turn, adding a spatial memory challenge on top of the content memory.

## Scoring / Victory Conditions

The game ends when the last search card has been won. Each player adds up the point values on their collected search cards. The player with the most points wins.

## Special Rules & Edge Cases

- Players should try to remember both what is under each pyramid AND where pyramids have shifted to.
- The sliding mechanism means treasures change position relative to the board, making pure memory insufficient.
- Some search cards may depict treasures that have already been found, requiring special handling per variant rules.
