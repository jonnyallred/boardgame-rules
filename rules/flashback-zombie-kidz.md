## Overview

Flashback: Zombie Kidz is a cooperative picture-based investigation game set in the Zombie Kidz universe. Players travel through frozen-in-time "Memories" — illustrated scenes viewed from different perspectives — to discover clues, solve mysteries, and ultimately save the world from a zombie invasion. The game consists of 3 chapters (Memories) played in order, with additional puzzles and tools (Gizmos) unlocked as you progress.

## Components

- 80 large Story Cards (numbered, with Memory-specific colored backs)
- 30 small Special Cards
- 2 Envelopes (sealed, opened during play when instructed)
- 1 Surprise Box (sealed)
- Mystery Booklet (sealed, opened after completing all 3 Memories)
- Rulebook with solutions

## Setup

1. Start with Memory 1.
2. Take Memory cards 01 to 20 (yellow backs) and place them in a face-down pile on the table.
3. Keep the decks of small and large Special cards nearby.
4. Leave the envelopes in the box.
5. Read the introduction for Memory 1 aloud, then flip card 01.

## Turn Structure

There are no formal turns. Players cooperatively explore the Memory by flipping cards.

### Exploring a Memory

1. Examine the current visible card(s) carefully. Each card shows a scene from a particular point of view.
2. Identify numbered points of view on the card — these correspond to other Story Cards you can flip to see what that character, animal, or object sees.
3. Flip the corresponding numbered card and place it in clear view on the table.
4. Take time to observe and understand each scene before revealing more cards.

### Special Cards

- When you see a **small Special card symbol** (e.g., a magnifying glass with a number), flip the matching small Special card from the Special deck.
- **Large Special cards** require a Gizmo to be revealed. Do not flip a large Special card without using the appropriate Gizmo.

### Mysteries

- Cards marked with **?** are Mysteries to solve. Read them aloud when discovered.
- Discuss and formulate theories as a team.
- You are not expected to have answers until the end of the Memory.

## Actions

### Solving Mysteries

At the end of each Memory (when all cards have been flipped):
1. Reread each Mystery.
2. Give your team's final answer for each.
3. Check solutions at the end of the rulebook.
4. For each correct answer, tick the brain box.
5. If all brains are ticked, you have played a perfect game.

### Gizmos

- Hidden objects discovered during play that have special properties.
- Used to reveal large Special cards and uncover hidden information in future Memories.
- Never open an envelope without getting permission from the game first.

### Progressing Through Memories

- Memory 1 (yellow cards 01-20): Investigate how zombies infiltrated the schoolyard.
- Memory 2 (new card set): Explore the school at night with traps and hiding spots.
- Memory 3 (new card set): Discover the zombie cook's plan and save the world.
- After completing all 3 Memories, the Mystery Booklet unlocks additional riddles using all Gizmos across all Memories.

## Scoring / Victory Conditions

This is a cooperative investigation game without traditional scoring. Success is measured by:
- Solving all Mysteries correctly (tick all brain boxes for a perfect game).
- Completing all 3 Memories in sequence.
- After all Memories: solving the Mystery Booklet's additional puzzles using Gizmos and clues from across all three Memories.

## Special Rules & Edge Cases

- **Cooperative Play**: Players should help each other, discuss observations, and share theories openly.
- **Pacing**: The game strongly advises slowing down and carefully observing each card before revealing the next. Rushing leads to missed clues and difficulty solving Mysteries.
- **Card Reveal Order**: You may reveal cards whose numbers you have found on already-visible cards in any order you like.
- **Incomplete Information**: When you discover a Mystery, you likely will not have enough information to solve it yet. This is expected — gather more clues as you progress.
- **Sealed Content**: Envelopes and the Surprise Box contain new game elements. Only open them when explicitly instructed by the game.
- **Replayability**: After completing the third Memory, new rules allow you to use all Gizmos across all Memories and solve new hidden puzzles using blue-backed cards.
- **The Keypad**: A meta-puzzle where correct Mystery answers correspond to numbers that form a symbol, which unlocks additional content.

## Player Reference

### Game Flow
1. Set up the current Memory's card pile
2. Flip the starting card (01 for Memory 1)
3. Explore by flipping numbered cards visible in the scenes
4. Reveal Special cards when their symbols appear
5. Read Mysteries aloud when discovered (marked with ?)
6. After all cards are revealed, solve all Mysteries
7. Check solutions and mark correct answers
8. Proceed to next Memory

### Key Rules
- Do not flip large Special cards without the required Gizmo
- Do not open sealed envelopes or boxes until instructed
- Take time to observe each scene carefully before moving on
- Discuss theories openly — this is a cooperative game
